# User Flow

## 1. Mentee
- Register/Login
- Browse mentors by field
- View sessions & enroll
- Attend session (in-person/Zoom)
- Leave feedback

## 2. Mentor
- Register/Login
- Create sessions
- View enrollments
- Receive feedback

## 3. Admin (optional)
- Approve mentors
- View all users/sessions
- Analytics dashboard (future idea)
